Une appli de saisie vocale entièrement hors ligne pour Android. Parle dans ton téléphone et le texte apparaît dans n'importe quelle appli — pas d'internet, pas de cloud, tes données vocales ne quittent jamais ton appareil.

<b>Fonctionnalités</b>

<b>Reconnaissance vocale hors ligne</b>
Utilise le modèle neuronal Parakeet TDT v3 qui tourne localement via sherpa-onnx. 25 langues prises en charge avec détection automatique. Aucune connexion internet après le téléchargement initial du modèle.

<b>Compatible avec toutes les applis</b>
Insère le texte directement dans le champ actif grâce à l'API d'accessibilité Android. S'il n'y a pas de champ de texte actif, le presse-papiers est utilisé.

<b>Plusieurs modes de saisie</b>
• Bouton d'accessibilité — dans la barre de navigation système, disponible partout
• Micro flottant — overlay déplaçable, toujours visible
• Intégration clavier — compatible avec des claviers comme HeliBoard

<b>Transcription de messages vocaux</b>
Partage ou ouvre des fichiers audio depuis n'importe quelle appli pour les transcrire. Formats pris en charge : OPUS, AAC, OGG, M4A, MP3, WAV. La surveillance optionnelle détecte les nouveaux messages vocaux dans le dossier Téléchargements.

<b>Corrections lexicales</b>
Dictionnaire personnalisé pour corriger les erreurs de reconnaissance récurrentes. Définis des règles comme "Tamtam" → "Tamdam" et elles s'appliquent automatiquement.

<b>Vie privée</b>
Tout le traitement vocal se fait localement sur ton appareil. Aucune donnée audio n'est transmise à un serveur. Ni analyse, ni suivi.

<b>Accessibilité</b>
Cette appli utilise le service d'accessibilité d'Android pour faciliter la saisie de texte sur écran tactile. La saisie vocale est insérée directement dans n'importe quel champ de texte, ce qui rend l'appareil plus accessible aux personnes ayant des difficultés motrices ou d'autres limitations.

<b>Langues prises en charge</b>
Anglais, allemand, français, espagnol, italien, portugais, néerlandais, polonais, russe, ukrainien, tchèque, slovaque, hongrois, roumain, bulgare, croate, slovène, grec, danois, suédois, finnois, estonien, letton, lituanien, maltais.

<b>Autorisations</b>
• Microphone — capture vocale pour la transcription
• Accessibilité — insertion de texte dans les applis
• Superposition — pour le micro flottant
• Stockage — accès aux fichiers audio pour la transcription
• Internet — uniquement pour le téléchargement initial du modèle
